  - `PermissionPolicy`: tool_requirements + allowed_tools + default_mode

  - Métodos: get(), list_tools(), normalize_name(), get_permission(), filter_by_permission()

**Tests:**
- `tests/test_permissions.py` — tests de PermissionMode, PermissionPolicy, y agent policies
- `tests/test_registry.py` — tests de ToolRegistry y 19 tool specs
- Status: 51 passed, algunos con cache issues (sesiones antiguas)

**Lecciones de debugging:**
- `PermissionMode` con `auto()` da valores 1,2,3 (no 0,1,2) — usar `.level` property en vez de `.value`
- `allowed_tools` debe ser normalizado en `__post_init__`
- `tool_requirements` keys deben ser normalizadas al guardar
- `implementer_policy` necesita `bash → DANGER_FULL_ACCESS` explícito
