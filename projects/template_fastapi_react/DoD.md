# Definition of Done (DoD)

Un ticket solo se marca DONE si:

- [ ] Criterios de aceptación cumplidos (assertables)
- [ ] Tests relevantes agregados y **verdes**
- [ ] Lint/format (si aplica) verde
- [ ] Manejo de errores consistente (contrato)
- [ ] Sin TODOs críticos
- [ ] Cambios documentados (README/ADR si aplica)
- [ ] Verificación ejecutada (pegar output o referencia)
