# Perfil: React (TypeScript)

## Objetivo
Frontend React con tests y build reproducible.

## Convenciones
- Componentes presentacionales vs contenedores
- API client centralizado
- Manejo consistente de estados: loading/empty/error

## Comandos (placeholder)
- Tests: `pnpm -s test`
- Lint: `pnpm -s lint`
- Build: `pnpm -s build`

## Estructura sugerida
- `frontend/src/pages`
- `frontend/src/components`
- `frontend/src/lib/api`
- `frontend/src/state`
- `frontend/src/test`
